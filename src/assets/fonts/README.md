# Gilroy Font Files

Для корректной работы шрифта Gilroy необходимо скачать следующие файлы:

## Необходимые файлы:

- `gilroy-light.woff2` (font-weight: 300)
- `gilroy-light.woff` (fallback для старых браузеров)
- `gilroy-regular.woff2` (font-weight: 400)
- `gilroy-regular.woff` (fallback)
- `gilroy-medium.woff2` (font-weight: 500)
- `gilroy-medium.woff` (fallback)
- `gilroy-semibold.woff2` (font-weight: 600)
- `gilroy-semibold.woff` (fallback)
- `gilroy-bold.woff2` (font-weight: 700)
- `gilroy-bold.woff` (fallback)

## Где скачать:

1. **Официальный сайт**: https://www.fontsquirrel.com/fonts/gilroy
2. **Google Fonts альтернатива**: Inter (уже настроен как fallback)
3. **CDN**: fonts.cdnfonts.com (может быть недоступен)

## Конвертация:

Если у вас есть TTF/OTF файлы, конвертируйте их в WOFF2/WOFF:

- https://convertio.co/ttf-woff2/
- https://cloudconvert.com/ttf-to-woff2

## Размещение:

Все файлы должны быть в папке `src/assets/fonts/`

После добавления файлов шрифт будет загружаться локально и работать стабильно.
